# How to Install ESign iOS on iPhone and iPad

This guide explains how to install ESign iOS using AppleJr.

## Requirements

- iPhone or iPad
- Safari browser
- Internet connection
- AppleJr DNS Profile
- Working certificate if required

## Installation Steps

1. Open Safari
2. Visit AppleJr website
3. Install the AppleJr DNS Profile
4. Go to the ESign section
5. Tap Install ESign
6. Wait until the app appears on your home screen
7. Open Settings
8. Go to General > VPN & Device Management
9. Trust the developer profile
10. Open ESign and start signing IPA files

## Troubleshooting

If ESign cannot be installed, try:

- Reinstalling the DNS profile
- Checking your internet connection
- Trying another available ESign link
- Restarting your iPhone
- Checking the AppleJr update page
