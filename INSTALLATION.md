# AppleJr Installation Guide

This guide explains how to use AppleJr to install IPA files on iPhone and iPad.

## Step 1: Open AppleJr Website

Open Safari on your iPhone or iPad and visit:

https://applejr.net

## Step 2: Install AppleJr DNS Profile

Go to the DNS section and install the AppleJr DNS Profile.

After downloading the profile:

1. Open Settings
2. Tap Profile Downloaded
3. Tap Install
4. Enter your passcode
5. Tap Install again

## Step 3: Install IPA Installer

Choose one of the available IPA installer apps:

- ESign
- Scarlet
- KSign

Install the app from the AppleJr website.

## Step 4: Trust Developer Profile

After installation:

1. Open Settings
2. Go to General
3. Go to VPN & Device Management
4. Tap the developer profile
5. Tap Trust

## Step 5: Import Certificate

Open the IPA installer app and import the certificate if required.

## Step 6: Sign and Install IPA

Import your IPA file, sign it using the selected certificate, and install it on your iPhone or iPad.
