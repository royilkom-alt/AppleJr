# How to Use AppleJr

AppleJr helps users find iOS sideloading tools, IPA installation guides, DNS profile guides, and troubleshooting tutorials.

## Main Features

## 1. IPA Installer

Use AppleJr to install IPA signing apps such as ESign, Scarlet, and KSign.

## 2. DNS Profile

Use the AppleJr DNS Profile before installing sideloading apps.

## 3. Certificate Guide

AppleJr provides certificate-related guides for IPA signing apps.

## 4. Troubleshooting

AppleJr includes guides for common iOS errors such as:

- Unable to Install App
- Unable to Verify App
- App Integrity Could Not Be Verified
- Revoked Certificate
