# AppleJr - iOS IPA Installer, ESign, Scarlet, KSign & DNS Installation Guide

AppleJr is an iOS guide hub for iPhone and iPad users who want to learn how to install IPA files using tools like ESign, Scarlet, KSign, and AppleJr DNS Profile.

This repository contains the official AppleJr installation guide, usage tutorial, FAQ, and troubleshooting documentation.
